Build mode
Click a faceSelect a 1×1 rect on its plane
Drag from a faceSelect a rect on its plane — overhang into air is fine
=Extrude the faces present in the rect one layer outward
− / DeleteCarve one layer of the rect's footprint (keeps marching through air)
OReset the corner offsets of the selected rect's faces
EscClear selection

Sculpt mode — tools
M / B / FSelect tool / Smooth brush / Draw brush
Brush dragPaint over the radius (sidebar sliders set radius & strength)
Alt + Draw brushPull the surface in instead of out
With "may add/remove voxels" on, brushes reshape the voxels themselves.
Sculpt mode — Select tool
ClickSelect a corner (only visible corners show)
Click a selected cornerDrag to move it (Alt = no snap)
DragBox select (Shift = add to selection)
Ctrl/Cmd+clickSelect the shortest edge path from the last-picked corner
X / Y / ZConstrain movement to an axis (Shift = plane ⊥ axis; same key clears)
= / −Nudge selection along the constrained axis, toward / away from camera
HSmooth selected corners toward their neighbors
U / JInflate / deflate selected corners along their normal
NNoise — organic jitter on selection
OReset selected corner offsets
All corner offsets are hard-clamped to ±½ cell on every axis.
